# sydingto
A simple custom UI Java Desktop with Swing

## Quick Installation
1. Clone the repository
```bash
git clone https://github.com/satriaajiputra/sydingto.git
```
2. Open project using ``netbeans 8.2``
3. Run project by pressing ``F6``
4. Happy coding
